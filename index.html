🚑 救急英語トレーナー 音声で「通じる」を体感

10症候別に、救急現場で必要な情報を英語で聞き出す練習。音声認識で意味が伝われば「通じた！」と判定します（多少の発音差はOK）。まず症候を選んでください。

　通じた 0 / 0
🎤 を押して英語で話し、話し終えたら ■ を押す。キーワードが伝われば成功です（語順や細かい発音が違ってもOK）。
⏱ 話し終えると自動で停止します（約2秒の無音で確定）。文中の短いポーズでは切れないので2文でも続けて言えます。途中で止めたい時は ■ を押してください。
※ マイク利用には Chrome / Edge と localhost または https での実行が必要です。
